LeetCode Exercises
===================
Ce référentiel contient mes solutions pour les exercices LeetCode. J'utilise ce référentiel pour garder une trace de mes progrès, partager mes solutions et me connecter avec d'autres personnes intéressées par le développement logiciel et l'algorithme.

Structure du référentiel
------------------------
Le référentiel est organisé par numéro de problème LeetCode, chaque problème ayant son propre dossier. Chaque dossier de problème contient le code de ma solution et une brève explication de l'approche utilisée.

```
LeetCode/  
├── 1. Problem/  
│   ├── solution.c
│   └── README.md  
├── 2. Problem/  
│   ├── solution.c  
│   └── README.md   
└── ...
```


Remarque
--------
Il est important de comprendre que LeetCode interdit explicitement le partage des solutions de problèmes pendant les compétitions actives. Assurez-vous de respecter les règles et les politiques de LeetCode lors de l'utilisation de ce référentiel.

N'hésitez pas à explorer ce référentiel et à me contacter si vous avez des questions ou des suggestions. Bonne programmation
